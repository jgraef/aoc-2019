# Day 12

This takes really long!

You can play the game! Run:

```bash
cargo run --release --bin arcade_game --feature arcade_game
```

### Controls

 - `A`: *left*
 - `B`: *right*
 - `J`: *auto pilot*
 - `ESC`: *exit*
